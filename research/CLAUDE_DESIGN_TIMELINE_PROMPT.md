### Visual Details

**Timeline Line**:
- Vertical, center-aligned
- Subtle glow (Sacred Flame purple-gold gradient)
- Grows/reveals as user scrolls (scroll-triggered animation)
- Small glowing dots at each card connection point

**Cards**:
- Glass-morphism on dark background
- Default: 60% opacity, date prominent, title + one-liner visible
- Hover: 100% opacity, glow effect, gap badge appears, file path in monospace
- Click/expand: Full detail text, proof section, verification paths
- Color-coded left border matching theme (purple/yellow/blue/red/green)

**Gap Badges**:
- Small pill-shaped badges on each card
- Show the "ahead of mainstream" metric
- Subtle animation (pulse or shimmer)

**Scroll Behavior**:
- Cards appear with staggered animation as line grows
- Each card slides in from alternating sides (left/right)
- Special cards (Sept 12, Final message) have unique entrance animation
