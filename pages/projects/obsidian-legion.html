◆ THE ARCHITECTURE

Five layers. One vault. Four agents sharing memory.

Each layer has a distinct role and a distinct cost. The router asks the cheapest layer first and escalates only when it must. 71× fewer tokens than naive retrieval.

#
layer · tech
what it does
L0
Knowledge graph→ Graphify
Typed edges between entities. The map the rest of the layers read. 71× fewer tokens than naive retrieval.
# resolve without loading the note
graph.neighbors("alexko", depth=2)
L1
Direct vault read→ CLI
Instant, free, local. The filesystem is first-class memory. Grep is a query language.
# zero tokens spent
legion read vault/*.md --grep "charter"
L2
LLM wiki compiler→ Karpathy
Compile, don't retrieve. The vault pre-bakes its own summaries. Context windows stay small; answers stay fast.
# once at compile time
legion wiki compile --since yesterday
L3
Vector search→ Qdrant
Semantic similarity as a last resort. Used when the graph has no edge yet. Every hit writes a new edge for next time.
# fallback path only
legion search "that film about craft"
L4
MCP Server→ native
Every agent talks to it natively. One vault, four minds, zero sync conflicts. Claude, Codex, Gemini, local.
# any MCP-capable agent
legion serve --mcp :4040
◆ FEATURES

The commands you'll actually run.

Legion is a CLI first. Everything is a subcommand. Every subcommand is idempotent. You can wake up tomorrow and grep your way back into the state you left.

F·01 · HANDSHAKE

Boot the crew with one line.

Starts the MCP server, warms the graph, opens the bridges. Agents boot in under a second.

$ legion handshake
# ◇ graphify  ok  · 12,844 nodes
# ◇ qdrant    ok  · 32,883 vectors
# ◇ mcp       ok  · :4040 listening
# ◇ bridge(4) ok  · claude · codex · gemini · local
F·02 · COMPILE

The wiki bakes itself overnight.

Karpathy-style compiled summaries. You wake up; yesterday is already condensed into a linked article.

$ legion wiki compile --since 24h
# 47 notes → 1 article · 2,112 → 180 tokens
# wrote /vault/wiki/2026-04-22.md
F·03 · ASK

Query the vault like a colleague.

Router picks the cheapest capable layer. You never specify which one. Graph first, files second, vectors last.

$ legion ask "what did we decide about scanlines?"
# L0 graph hit · 0 tokens · answered in 41ms
"scanline intensity tweak stays at 0.35"
F·04 · ROAST

Adversarial review on demand.

Pick an agent. Point it at a note. Get the argument you were hoping nobody would make.

$ legion roast --agent hal --note prd.md
# hal-tars: "line 47 asserts 'never'. prove it."
# hal-tars: "spec lacks a rollback path. rewrite §3."
